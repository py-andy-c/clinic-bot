LINE 訊息統計說明

此頁面顯示 LINE 訊息使用統計，包括推播訊息和 AI 回覆訊息的使用情況。

數據期間：顯示過去 3 個月及當月的數據。當月數據可能仍會變動。

格式：數據以「數量(百分比%)」格式顯示，百分比為四捨五入後的整數。

LINE 推播訊息

LINE 推播訊息由 LINE 平台收費，診所需自行負擔相關費用。

若 LINE 訊息配額用盡，系統將無法發送推播訊息，但預約系統仍可正常運作。

AI 回覆訊息

AI 回覆訊息不會消耗 LINE 訊息配額。
